# AWS S3 Static Website Hosting Project

## Project Overview

This project demonstrates how to host a static website using Amazon S3. A TemplateMo website template was uploaded to an S3 bucket and configured for static website hosting.

## Steps Performed

### Step 1

Searched for a TemplateMo website template using a web browser.

### Step 2

Signed in to the AWS Management Console.

### Step 3

Searched for Amazon S3 from the AWS services dashboard.

### Step 4

Created a new S3 bucket named:

`s3practically`

### Step 5

Opened the **Permissions** tab of the bucket.

### Step 6

Created and configured a bucket policy to allow public access to the website files.

### Step 7

Opened the **Properties** tab.

### Step 8

Enabled **Static Website Hosting**.

### Step 9

Specified:

`index.html`

as the index document.

### Step 10

Uploaded the TemplateMo website files (templatenovapay) to the bucket.

## Result

The static website was successfully hosted on Amazon S3 and became accessible through the S3 website endpoint URL.

## Technologies Used

* Amazon S3
* AWS Management Console
* Static Website Hosting
* Bucket Policy
* HTML, CSS, and JavaScript
